# Coolvetica Light

To use Coolvetica Light on the landing page, add the font file here:

- **Preferred:** `Coolvetica-Light.woff2` (or `.ttf`)
- **Alternative:** Download [Coolvetica](https://www.dafont.com/coolvetica.font) (e.g. “Coolvetica Rg” = Regular) and save as `Coolvetica-Light.woff2` or `Coolvetica-Light.ttf`, or use the OTF and we can add support.

If no file is present, the site will fall back to the default sans-serif font.
